Datenschutzerklärung

Stand: Dezember 2024

Diese Datenschutzerklärung gilt für die App MediaTechManager (im Folgenden "App" genannt).

Zusammenfassung: MediaTechManager speichert alle Daten ausschließlich lokal auf Ihrem Gerät. Es werden keine Daten an Server übertragen oder mit Dritten geteilt.

1. Verantwortlicher

Verantwortlich für die Datenverarbeitung ist:

Manuel Reinert
E-Mail: support@mediatechmanager.app

2. Welche Daten werden erhoben?

Die App erhebt und verarbeitet folgende Daten ausschließlich lokal auf Ihrem Gerät:

  • Equipment-Daten: Informationen zu Ihren Geräten (Name, Seriennummer, Kategorie, Standort, Zustand)
  • Kundendaten: Name, Kontaktinformationen, Firmenname (falls eingegeben)
  • Verleihdaten: Verleihzeiträume, Unterschriften, Zustandsberichte
  • Fotos: Equipment-Bilder (nur wenn Sie diese hinzufügen)

3. Geräteberechtigungen

Die App kann folgende Gerätefunktionen nutzen:

  • Kamera: Zum Scannen von QR-Codes und Aufnehmen von Equipment-Fotos. Die Kamera wird nur aktiviert, wenn Sie diese Funktion nutzen. Bilder werden nur lokal gespeichert.
  • Face ID / Touch ID: Zur optionalen Absicherung der App mit biometrischer Authentifizierung. Biometrische Daten werden niemals von der App gespeichert oder übertragen - die Authentifizierung erfolgt über das iOS-System.
  • Fotobibliothek: Zum Speichern und Laden von Equipment-Bildern. Zugriff erfolgt nur auf Ihre Anfrage.

4. Datenspeicherung

Lokale Speicherung: Alle Daten werden ausschließlich lokal auf Ihrem Gerät in einer verschlüsselten Datenbank gespeichert. Es erfolgt keine Übertragung an externe Server.

Die Daten verbleiben auf Ihrem Gerät, bis Sie:

  • Die Daten manuell löschen
  • Die App deinstallieren
  • Ein Backup exportieren (dann als lokale Datei)

5. Datenweitergabe an Dritte

Es erfolgt keine Weitergabe von Daten an Dritte.

Die App:

  • Enthält keine Werbung
  • Enthält keine Analytics- oder Tracking-Dienste
  • Überträgt keine Daten an Server
  • Teilt keine Daten mit Dritten

6. Ihre Rechte

Sie haben folgende Rechte bezüglich Ihrer Daten:

  • Auskunft: Alle Daten sind direkt in der App einsehbar
  • Berichtigung: Sie können alle Daten jederzeit in der App bearbeiten
  • Löschung: Sie können einzelne Einträge oder alle Daten löschen
  • Export: Sie können Ihre Daten als Backup exportieren

7. Datensicherheit

Ihre Daten werden durch folgende Maßnahmen geschützt:

  • Lokale Speicherung ohne Serverübertragung
  • iOS-Datenverschlüsselung
  • Optionale PIN-/Face ID-Sperre
  • Keine Cloud-Synchronisation
